# Craftify

Some useful tools for your daily work

## Commands

- `to-main` - Translate text from any language to Main Language or Main Language to Second Language
- `to-bg` - Translate text any language To Bulgarian
- `to-ua` - Translate text any language To Ukrainian
- `fix-text` - Fix errors in text
- `summarize` - Summarize text
- `explain` - Explain text
